##IDEA Dispute Resolution Data

This is an analysis of publicly available IDEA state level dispute resolution 
data. The analysis uses the Part B dispute resolution data available on the [US Department of Education's website](https://www2.ed.gov/programs/osepidea/618-data/state-level-data-files/index.html#bdr). The documentation for the data sets are kept as Word files in the [Data Document section](https://www2.ed.gov/programs/osepidea/618-data/collection-documentation/index.html).
